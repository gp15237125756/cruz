private String findAnnotationName(Class<?> clazz) {
    Extension extension = (Extension)clazz.getAnnotation(Extension.class);
    if (extension == null) {
        String name = clazz.getSimpleName();
        if (name.endsWith(this.type.getSimpleName())) {
            name = name.substring(0, name.length() - this.type.getSimpleName().length());
        }

        return name.toLowerCase();
    } else {
        return extension.value();
    }
}

injectExtension方法解析

injectExtension作用是检查SPI实现类中的set方法 实现set注入

injectExtension遍历实现类的所有set方法,object = this.objectFactory.getExtension(pt, property);这句话调用AdaptiveExtensionFactory的getExtension方法，分别使用SpiExtensionFactory获取Adaptive类型的扩展
以及使用SpringExtensionFactory获取Spring方式实现的扩展

也就是说，如果用spi实现依赖的话，只有加上@Adaptive注解，才能获得依赖属性的扩展实现，实现set注入；
显然injectExtension方法，内部使用递归方式实现set注入

JavaScript中Promise使用

2019-01-24

定义

Promise对象用于包装异步函数执行结果，以便用同步的方式处理其结果。

promise 包含3种状态：

pending： 初始状态

fulfilled: 完成状态

rejected: 失败状态

链式调用方法

then()返回一个Promise对象 或者构造一个空白的Promise对象（如Promise.resolve()返回一个初始状态是fulfilled的延迟对象）

处理延迟对象数组

Promise.all([]) 参数传入Promise对象数组 then中得到结果的数组
Promise.all() takes an iterable (array or array-like list) of promise objects, and waits until all of those promises have been fulfilled before moving on to any then() method attached to it. The then() method is passed an array of returned values from each promise.
